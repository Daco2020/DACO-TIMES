# 다코 타임즈
- 외부 뉴스API를 활용한 최신 뉴스 검색, 조회 서비스
- [다코 타임즈 바로가기](https://daco-times.netlify.app/)

# 기능
- 최신 뉴스를 조회합니다.
- 카테고리별 뉴스를 조회할 수 있습니다.
- 키워드를 검색할 수 있습니다.
- 페이지네이션을 구현하였습니다.
- 돋보기를 두번 누르면 검색창이 사라집니다.(토글)
